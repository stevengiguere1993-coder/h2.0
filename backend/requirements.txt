# OCR — extraction depuis images (screenshots de tableaux Excel,
# photos de fiches MLS, captures d'écran) et PDFs scannés. Stack
# 100% locale, sans aucun LLM externe :
#   - pytesseract : binding Python vers le binaire Tesseract
#   - Pillow      : manipulation d'images (déjà tiré par d'autres
#                   paquets, on l'épingle ici explicitement)
#   - pdf2image   : convertit un PDF en images PIL page par page
#                   (fallback OCR pour PDFs scannés sans couche texte)
#   - pillow-heif : support des photos HEIC/HEIF iPhone (iOS par
#                   défaut depuis iOS 11). Sans ça, Pillow ne lit
#                   pas les .heic et l'OCR ignore les photos prises
#                   directement avec un iPhone.
#
# IMPORTANT : ces paquets Python dépendent de binaires système
# (tesseract-ocr, tesseract-ocr-fra, poppler-utils) installés via
# le fichier `Aptfile` à la racine du backend. Sans ces binaires,
# pytesseract et pdf2image lèvent une exception au premier appel.
